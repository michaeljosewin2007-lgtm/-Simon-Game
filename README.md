# Simon Game

A browser-based implementation of the classic Simon Memory Game built using HTML, CSS, JavaScript, and jQuery.

## Overview

The Simon Game challenges players to remember and repeat an increasingly long sequence of colors. The game becomes progressively harder as the player advances through levels.

## Features

- Random sequence generation
- Multiple difficulty levels through increasing sequence length
- Sound effects for each button
- Interactive animations
- Game over detection and restart functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Concepts Practiced

- Arrays and Sequence Management
- Event Handling
- jQuery DOM Manipulation
- Game Logic Development
- Audio Integration

## Learning Outcome

This project helped me understand how to manage application state, generate random patterns, and build interactive games using JavaScript and jQuery.

## How to Play

1. Press any key to start the game.
2. Watch the color sequence carefully.
3. Repeat the sequence by clicking the buttons.
4. Advance through levels by reproducing the sequence correctly.
5. The game ends when an incorrect button is selected.
